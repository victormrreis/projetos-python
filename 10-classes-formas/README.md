# Classes — Formas Geométricas

Cálculo de áreas usando polimorfismo. Classe base `Forma` com filhas `Quadrado`, `Retangulo` e `Circulo`, cada uma com seu próprio cálculo de área.

## Conceitos aplicados

- Polimorfismo (mesmo método, comportamentos diferentes)
- Herança
- Iteração sobre lista de objetos com `for`
- F-strings com formatação numérica

## Como usar

```bash
python classe_formas.py
```
