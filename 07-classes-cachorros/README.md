# Classes — Cachorros

Duas classes que interagem: `Cachorro` com atributos e comportamentos, e `Dono` que gerencia uma lista de cachorros.

## Funcionalidades

- Cachorro: latir, comer, brincar, fazer aniversário, ver status
- Sistema de fome com limites (0-10)
- Dono: adotar, listar cachorros, alimentar todos, passear com todos

## Conceitos aplicados

- Múltiplas classes no mesmo arquivo
- Listas de objetos como atributos
- Classe recebendo objetos de outra classe
- Métodos que modificam estado com validação de limites
- `for` percorrendo listas de objetos

## Como usar

```bash
python classe_cachorros.py
```
