# Classes — Aluno

Primeiro projeto usando Programação Orientada a Objetos. Classe que representa um aluno com notas.

## Funcionalidades

- Criar alunos com nome
- Adicionar notas
- Calcular média automaticamente
- Verificar situação (Aprovado/Reprovado)

## Conceitos aplicados

- `class`, `__init__`, `self`
- Métodos que modificam atributos
- `return` em métodos
- Método chamando outro método (`situacao` chama `calcular_media`)
- Listas como atributos de objeto

## Como usar

```bash
python classes_aluno.py
```
