# Calculadora de Média

Programa que calcula a média de 3 notas e informa se o aluno foi aprovado ou reprovado.

## Conceitos aplicados

- Funções com `def` e `return`
- Parâmetros e argumentos
- Condicionais (`if/else`)
- F-strings com formatação (`.2f`)

## Como usar

```bash
python calculadora_media.py
```

O programa pede 3 notas e exibe a média e a situação (Aprovado se >= 7, Reprovado se < 7).
