# Carrinho de Compras Interativo

Sistema de compras no terminal com catálogo de produtos, menu interativo e cálculo de total.

## Funcionalidades

- Ver itens no carrinho com preços e total
- Adicionar produtos de um catálogo
- Remover itens pelo nome
- Encerrar a compra

## Conceitos aplicados

- Dicionários aninhados (nome + preço)
- Listas de dicionários
- Loops aninhados (`while` dentro de `while`)
- `break` e `continue` para controle de fluxo
- `casefold()` para comparação de strings
- `sum()` com generator expression para calcular total
- Limpeza de tela com `os.system()`

## Como usar

```bash
python carrinho_de_compras.py
```
