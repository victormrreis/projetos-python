# Conversor de Temperatura

Programa que converte temperaturas entre Celsius e Fahrenheit.

## Conceitos aplicados

- Funções com múltiplos parâmetros
- `return` para devolver valores
- `strip()` e `casefold()` para normalização de entrada
- Condicionais

## Como usar

```bash
python conversor_temperatura.py
```

O programa pede a temperatura e a direção da conversão (C→F ou F→C).
